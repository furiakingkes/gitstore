# GitStore — furiakingkes

Este repositório é criado automaticamente pelo **GitStore** para armazenar os assets visuais da sua organização.

## Estrutura

```
gitstore/
├── gitstore_readme.md   # Este arquivo
└── assets/              # Imagens, logos, banners e outros recursos visuais
    ├── avatar/
    └── banner/
```

## Por que este repositório existe?

O GitStore utiliza este repositório para armazenar o **avatar** e o **banner** da organização.
A descrição e demais informações são sincronizadas diretamente no perfil da organização no GitHub.

> Não exclua este repositório — ele é necessário para o funcionamento correto da organização no GitStore.
